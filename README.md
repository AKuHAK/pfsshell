
PFS (PlayStation File System) client for Linux/Mac/Unix
=======================================================

Information
===========

This is a tool to allow you to browse and send files to PFS filesystems.

Not supported
=============

> GUI (currently)
> 64-bit (if you are reading PFS filesystems, it might cause a segfault)
